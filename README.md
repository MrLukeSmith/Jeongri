# Jeongri

Jeongri (정리) is a Korean word that generally means to clean up, organize and rearrange. Which felt apt for a set of opinionated, and somewhat tailored, skills for organising and cleaning up development. 

![Logo](/Jeongri.jpg)

# Inspiration

Very much inspired by the incredible collection over at https://github.com/obra/superpowers, which is still a daily driver in my development toolkit. The more I used it, and the more I embraced agentic development, the more I was learning how I wanted the tools to work for me. Thus, Jeongri.
